Des Instructions
À l'aide de l'un de vos projets django précédemment créés qui utilise le modèle utilisateur , créez un CustomUseren étendant le modèle utilisateur à l'aide du fichier AbstractBaseUser.

Trouvez et suivez un didacticiel pour bien faire les choses. Vérifiez que vous pouvez vous connecter/déconnecter/enregistrer.
